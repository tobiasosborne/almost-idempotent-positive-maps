\step{1.1.1} E1 (left, l-term). l:=h\_\{p,b\}=p‚ąėb-P(p‚ąėb) where p=P(a\textasciicircum{}2) in A=Im P and b in A (def square-hole, def near-fixed-algebra). \step{1.1.2} E2 (left, h\_\{a,a\}-term). The product hole h\_\{a,a\}=a‚ąėa-P(a‚ąėa)=a\textasciicircum{}2-P(a\textasciicircum{}2) in Ker P (def square-hole). \step{1.2} RIGHT REDUCTION. With a,b,p as in node 1.1, set q:=P(b‚ąėa)=b‚ąėa-h\_\{b,a\} in A, h\_\{b,a\}=b‚ąėa-P(b‚ąėa) in Ker P (def square-hole). \step{1.2.1} E3 (right, h\_\{a,a\}‚ąė(b‚ąėa) term). h\_\{a,a\}=a\textasciicircum{}2-P(a\textasciicircum{}2) in Ker P (def square-hole). Uses node 1.1, node 1.2, node 1.3.1, GT-jordan-identity, GT-jordan-product, def near-fixed-algebra, def jordan-product. \steptag{validated} \\[6pt]
\step{1.3.1} SYMMETRIC AMBIENT JORDAN IDENTITY (exact).
