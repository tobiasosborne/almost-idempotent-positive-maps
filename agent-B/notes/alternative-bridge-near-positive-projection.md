# Alternative Bridge: Near-Positive Projection Stability

The contextual Effros-Stormer estimate is one route. A second route may be cleaner.

## Observation

For `Phi` unital positive with `||Phi^2-Phi||<=eta`, the spectral idempotent

```text
P = theta(2Phi-I)
```

satisfies

```text
P^2=P,  P(1)=1,  ||P-Phi||=O(eta).
```

Since `Phi` is positive, `P` is `O(eta)`-positive:

```text
x >= 0, ||x||<=1  =>  P(x) >= -C eta 1.
```

Thus the problem may reduce to:

## Near-Positive Projection Stability Conjecture

There are universal constants `C,delta0` such that if `P:B(H)_sa -> B(H)_sa` is a unital idempotent linear map with

```text
||P|| <= 1+delta
```

or more strongly

```text
x>=0, ||x||<=1 => P(x) >= -delta 1,
```

then there exists a unital positive idempotent `E:B(H)_sa -> B(H)_sa` with

```text
||E-P|| <= C delta^alpha
```

for some universal `alpha` (ideally `1`).

If true, Effros-Stormer applies exactly to `E`, and the product/range for `P` is close to the exact JC algebra from `E`.

This route would also bypass the current missing contextual estimate. Exact
Effros-Stormer does not use only the first insertion identity; it also proves a
null-ideal statement for holes generated by range products. Quantitatively, we
are missing estimates of the form

```text
h_{r,s}=r o s-P(r o s),
||P(h_{r,s}^2)|| small
```

or a bilinear version for `P(h_{r,s} o h_{u,v})`. A nearby positive idempotent
`E` would provide these estimates indirectly by comparing `P` with `E`.

## Classical Evidence

In the commutative case `l_infty^n`, a sidecar search found no counterexamples and reduced the question to:

```text
unital idempotent P on l_infty^n with ||P|| <= 1+delta
is O(delta)-close to a positive unital idempotent.
```

The exact case is true: norm-one unital projections on `C^n` are positive. Random and structured tests showed dimension-free behavior.

## Risks

- In noncommutative `B(H)_sa`, near contractive projections may be close to ranges that are not positive projection ranges with dimension-free constants; this is not known.
- If only approximate positivity is assumed for arbitrary maps, repairing the
  cone is impossible at `O(delta)` dimension-free in general: a spin-factor
  target gives epsilon-positive unital maps at distance `sqrt(epsilon)` from
  every positive unital map. See
  `agent-B/notes/subagent-positivity-rounding.md`.
- This does not automatically refute near-positive **projection** stability:
  idempotency requires a retraction onto its own range, a much stronger
  constraint than an arbitrary approximately positive map into a JB target.
  The open question is whether the spin-factor obstruction can be realized by
  an idempotent `P=iT` with `T i=id`.
- A sidecar check of this question found no plausible spin-factor upgrade. In
  the natural hidden-commutant construction, adding the signed sidecar
  perturbation while keeping `K|_J=0` makes an idempotent, but the positivity
  defect rises from `epsilon` to `Theta(sqrt(epsilon))`, matching its distance
  to the existing positive spin projection. The direct factor-through-classical
  construction also fails the retraction condition. See
  `agent-B/notes/subagent-near-positive-projection-stability.md`.
- This route may be as hard as the original approximate Jordan identity, but it packages the problem as a perturbation theorem for positive projections.

## Relation To Kitaev

Kitaev explicitly notes that an eta-idempotent UCP map need not be O(eta)-close to an idempotent UCP map. This alternative route asks for closeness to an idempotent **positive** map in the weaker UP setting; it may be easier, but the UCP counterexample warns against assuming it without proof.
