# Sidecar: Spin Factors And Near-Positive Projection Stability

## Question

Can the spin-factor obstruction to black-box positivity rounding be upgraded to
an idempotent counterexample?

The desired counterexample would be a unital idempotent

```text
P : B(H)_sa -> B(H)_sa
```

with range a concrete spin factor `i(J)`, such that `P` is `delta`-positive but
has distance much larger than `delta` from every unital positive idempotent.
Writing `P=iT`, this means

```text
T : B(H)_sa -> J,        T i = id_J.
```

The known rounding obstruction gives an `epsilon`-positive unital map
`T0: ell_infty^{2m}->J` at distance `sqrt(epsilon)` from positive maps, but it
does not impose this retraction constraint.

## Concrete Spin Retraction Setup

Represent the spin factor

```text
J = R 1 + span{s_1,...,s_m} subset B(K)_sa
```

using anticommuting symmetries `s_j`. In finite dimension, the trace-orthogonal
projection

```text
E0(x) = tau(x) 1 + sum_j tau(x s_j) s_j
```

is a unital positive idempotent onto `J` (Effros-Stormer Lemma 2.3 gives this
projection for spin factors).

Any idempotent onto the same concrete range has the form

```text
P = i(T0 + K),       K|_J = 0,
```

where `T0=i^{-1}E0`. Thus idempotency itself is easy; the issue is whether `K`
can carry the sidecar negative variation while preserving `delta`-positivity.

Dual form of the constraint: for every unit `a in R^m`, set

```text
s_a = sum_j a_j s_j,        p_a = (1+s_a)/2.
```

Since `T i = id`, the positive spin-face element `p_a` is mapped to the boundary
point `(1/2,a/2)` of the spin cone. Hence there is essentially no slack on this
face. If `K` has outward component in direction `a`, near-positivity forces

```text
sup_{0<=x<=1} [ a . K_vec(x) - K_0(x) - tau((1-s_a)x) ]_+ <= delta.
```

Equivalently, if `G_a` represents the functional
`a.K_vec - K_0`, then the negative trace mass of

```text
1 - s_a - G_a
```

must be at most `delta`. In particular, the compression of `G_a` to the support
`p_a` must have only `O(delta)` positive mass.

## Naive Hidden-Commutant Upgrade Fails

The most direct attempt is:

1. take a multiplicity representation `H=K tensor R`;
2. let the spin system act on `K`;
3. put the sidecar `2m` atoms in the commutant, using projections
   `q_{j,+}, q_{j,-}` on `R` with normalized trace `alpha=1/(2m)`;
4. define

```text
K_vec,j(x) = 2( tau(x (1 tensor q_{j,+}))
              - tau(x (1 tensor q_{j,-})) ).
```

Then `K|_J=0`, so `P=E0+iK` is a unital idempotent, and the restriction to the
commutative algebra generated by the `q_{j,pm}` has exactly the same sidecar
atom values:

```text
q_{j,+} -> (alpha,  2 alpha e_j),
q_{j,-} -> (alpha, -2 alpha e_j).
```

However the positivity defect is not `alpha`. For a unit vector `a`, the
operator testing the exposed spin face has eigenvalues

```text
1 - lambda - 2 mu,
lambda in {+1,-1},       mu in {+/- a_j}.
```

On the `lambda=+1` spin face, the negative contribution is `2 mu_+`. Therefore
the defect is at least

```text
alpha * sum_j |a_j|.
```

For `a=(1,...,1)/sqrt(m)` this is

```text
1/(2 sqrt(m)) = Theta(sqrt(alpha)),
```

not `alpha`. Thus the same construction is only
`Theta(sqrt(epsilon))`-positive, while its distance to the exact positive
projection `E0` is also `Theta(sqrt(epsilon))`. It gives no counterexample to
dimension-free projection stability.

Interpretation: the sidecar map spends the scalar budget once. The retraction
constraint already spends that budget on every spin face `p_a`, because
`T(p_a)` is a boundary point. Hidden vector variation in the same direction is
then detected by compressing to `p_a`, and the `l1` variation reappears.

## Direct Factor-Through-Classical Upgrade Also Fails

If one tries to write `T = T_sidecar M` with `M:B(H)_sa -> ell_infty^{2m}` a
positive unital measurement map, then `T i=id` would require, for each spin
generator,

```text
2 epsilon ( M(s_k)_{k,+} - M(s_k)_{k,-} ) = 1.
```

But each coordinate of a positive unital map to `ell_infty` is a state, so
`|M(s_k)_i| <= ||s_k|| = 1`. Hence the left-hand difference is at most
`4 epsilon`, impossible for the sidecar scaling
`epsilon=1/(2m)` when `m` is large.

This rules out the cleanest realization of the original signed-POVM example as
a retraction.

## Current Conclusion

I do not see a plausible spin-factor counterexample at the idempotent level.
The known sidecar obstruction relies on vector variation that is harmless for an
arbitrary approximately positive map, but the condition `T i=id` forces all spin
faces to be saturated boundary tests. In the natural concrete constructions,
those faces upgrade the defect from `epsilon` to `sqrt(epsilon)`, matching the
distance to the existing positive projection.

This does not prove near-positive projection stability. It only says that the
known spin-factor rounding obstruction does not transfer directly.

## Remaining Precise Test

A genuine spin-factor counterexample would need a perturbation `K` with

```text
K|_J = 0,        P=E0+iK idempotent,
```

and simultaneously:

1. for every unit `a`, the positive mass of the compression of
   `G_a = a.K_vec - K_0` to the spin face `p_a=(1+s_a)/2` is `O(delta)`;
2. on some commutative positive test family, the induced signed atom variation
   is still large enough to force distance `>> delta` from every positive
   unital map into `J`;
3. the lower bound is stable against comparison with arbitrary positive
   idempotents `E`, not only positive retractions onto the same spin factor.
   A useful reduction is: if `E` is positive and close to `P`, then
   `T0 E` is a positive unital map into `J` close to `T0 P=T`; hence any
   restriction of `T` that is far from positive maps gives the same lower bound.

The unresolved hard part is item 1 plus item 2 together. The commutant/PVM model
satisfies item 2 but violates item 1 at the `sqrt(delta)` scale.
