# Approximate Effros-Stormer Notes

This note records Agent B's current attempt to quantify the first step of the positive-map generalisation.

## Exact Input

Effros-Stormer, *Positive projections and Jordan structure in operator algebras*, Theorem 1.4:

If `P:A -> A` is a unital positive projection on a unital JC-algebra, then `P(A)` is a JC-algebra for the product

```text
r * s = P(r o s),    r,s in P(A),
```

where `o` is the ambient Jordan product. The key exact identity is Lemma 1.1:

```text
P(P(a) o P(b)) = P(a o P(b)).
```

The JW-support argument then gives the stronger identity

```text
P(x o y) = P(P(x) o P(y))
```

for `x,y` in the JW-algebra generated by `P(M)`, and this proves the Jordan identity for `*`.

## Approximate Setup

Let `V = B(H)_sa` with order-unit norm and Jordan product `a o b = (ab+ba)/2`.
Let `Phi:V -> V` be unital positive with

```text
||Phi^2 - Phi|| <= eta,       eta < 1/4.
```

Let

```text
Pi = theta(2 Phi - I)
```

by Banach-algebra functional calculus. Then

```text
Pi^2 = Pi,    Pi(1)=1,    ||Pi-Phi|| <= C eta.
```

Set `delta = ||Pi-Phi||`.
`Pi` is not necessarily positive, but it is `delta`-positive:

```text
a >= 0  =>  Pi(a) >= -delta ||a|| 1.
```

Define `R = Im Pi` and

```text
r * s = Pi(r o s).
```

## Proven/Mostly Proven Lemma: Approximate Lemma 1.1

There is a universal constant `C` such that for all `a,b in V`,

```text
|| Pi(Pi(a) o Pi(b)) - Pi(a o Pi(b)) ||
   <= C sqrt(eta) ||a|| ||b||.
```

### Proof Sketch

Fix a state `rho` on `B(H)_sa` and set `omega = rho o Phi`.
For `x in V`, define the seminorm

```text
||x||_omega^2 = omega(x^2).
```

Since `Phi` is unital positive, Jordan-Schwarz gives `(Phi x)^2 <= Phi(x^2)`.
Using `||Pi-Phi|| <= delta` and `||Phi^2-Phi|| <= eta`,

```text
||Pi x||_omega^2
 = rho Phi((Pi x)^2)
 <= rho Phi((Phi x)^2) + O(delta)||x||^2
 <= rho Phi^2(x^2) + O(delta)||x||^2
 <= ||x||_omega^2 + O(eta)||x||^2.
```

Thus `Pi`, although not positive, is an almost contraction in every seminorm `||.||_omega`.
Because `Pi` is an exact idempotent, its range and kernel are almost orthogonal in these seminorms:

```text
| omega(u o n) | <= C sqrt(eta) ||u|| ||n||
```

for `u in Im Pi`, `n in Ker Pi`.
Taking `u=Pi(a)` and `n=a-Pi(a)`, and comparing `rho Pi(z)` with `rho Phi(z)=omega(z)`, gives the displayed estimate.

## Gap: Stronger Generated-Algebra Identity

The approximate Lemma 1.1 alone does **not** yet prove the approximate Jordan identity for `*`.
The exact Effros-Stormer proof also uses the stronger identity

```text
P(x o y)=P(P(x)oP(y))
```

for `x,y` in the Jordan algebra generated by the range.

For the approximate theorem we need one of:

1. a quantitative support-projection argument proving, at least for low-degree words needed in the Jordan identity,

```text
|| Pi(x o y)-Pi(Pi(x)oPi(y)) ||
   <= C eta^alpha ||x|| ||y||
```

for `x,y` in the ambient Jordan algebra generated by `R`; or

2. a different proof of the Jordan identity defect for `r*s=Pi(r o s)` that avoids this stronger identity; or

3. a counterexample showing UP almost-idempotence is insufficient, forcing a stronger hypothesis.

This is currently the central technical bottleneck.

If `eta=0` and `Pi=Phi=P` is positive, the stronger identity is exactly Effros-Stormer Lemma 1.2(4), hence the range is a genuine JC-algebra.
