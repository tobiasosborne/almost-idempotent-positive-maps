# Partial Layer 2 Bridge: Square-Root Insertion

This is Agent B's current rigorous bridge result. It proves the first insertion estimate, but **does not yet prove** the approximate Jordan identity. An earlier draft of this note overclaimed the corollary; the correction is below.

## Setup

Let `V = B(H)_sa` with Jordan product

```text
x o y = (xy+yx)/2
```

and order-unit/operator norm. Let `Phi:V -> V` be unital positive and

```text
||Phi^2 - Phi|| <= eta,        eta < eta0.
```

Let

```text
P = theta(2 Phi - I)
```

by Banach-algebra functional calculus. Then

```text
P^2=P,       P(1)=1,       ||P-Phi|| <= c eta,       ||P|| <= 1+c eta.
```

Set `A = Im P` and define

```text
a * b = P(a o b),     a,b in A.
```

## Lemma 1: Almost-Contraction In State Seminorms

For every state `rho` on `B(H)` define the positive functional

```text
omega = rho o Phi
```

and seminorm

```text
||x||_omega^2 = omega(x^2) = rho(Phi(x^2)).
```

Then, for all `x in V`,

```text
||P x||_omega^2 <= ||x||_omega^2 + C eta ||x||^2.
```

Proof:

```text
rho Phi((Px)^2)
 <= rho Phi((Phi x)^2) + C eta ||x||^2
 <= rho Phi(Phi(x^2)) + C eta ||x||^2        [Jordan-Schwarz]
 <= rho Phi(x^2) + C eta ||x||^2.            [near idempotence]
```

## Lemma 2: Almost Orthogonality Of Range And Kernel

Let `u in Im P` and `n in Ker P`. Then for every state-seminorm above,

```text
| omega(u o n) | <= C sqrt(eta) ||u|| ||n||.
```

Proof:

Apply Lemma 1 to `u+t n`. Since `P(u+t n)=u`,

```text
||u||_omega^2 <= ||u+t n||_omega^2 + C eta ||u+t n||^2
```

for all real `t`. Expanding the Hilbert seminorm and optimizing `t` at scale
`sqrt(eta)||u||/||n||` gives the estimate.

## Lemma 3: Insertion Estimate

For all `x in V` and `b in A`,

```text
|| P(Px o b) - P(x o b) || <= C sqrt(eta) ||x|| ||b||.
```

Proof:

Let `n=x-Px in Ker P`. For any state `rho`,

```text
rho(P(Px o b)-P(x o b)) = -rho(P(n o b)).
```

Using `||P-Phi|| <= C eta`,

```text
|rho(P(n o b))|
 <= |rho(Phi(n o b))| + C eta ||n|| ||b||
 = |omega(n o b)| + C eta ||n|| ||b||.
```

Now apply Lemma 2 with `u=b` and `n=x-Px`. Supremising over states gives the operator-norm bound.

## What This Does Not Yet Prove

The estimate above controls products of a kernel vector with a range vector:

```text
P((I-P)x o b),       b in Im P.
```

The Jordan identity for `*` also exposes terms where a kernel vector is multiplied by an ambient monomial that is not known to lie in `Im P`.

Example: for `a,b in A`, the right-hand side contains

```text
P(P(a o a) o P(b o a)).
```

Using Lemma 3 once gives

```text
P(P(a o a) o P(b o a)) approx P((a o a) o P(b o a)),
```

but removing the remaining `P` from `P(b o a)` would require controlling

```text
P((a o a) o (I-P)(b o a)),
```

where `a o a` is not generally in `Im P`.

Thus Lemma 3 is only the approximate version of Effros-Stormer Lemma 1.1. The exact Effros-Stormer proof needs the stronger Lemma 1.2(4),

```text
P(x o y)=P(Px o Py)
```

on the Jordan algebra generated by the range. The approximate theorem still needs a quantitative replacement for that stronger identity.

## A Conditional Corollary

If, in addition to Lemma 3, one has a contextual insertion estimate

```text
||P(C[P(x)] ) - P(C[x])|| <= eps ||x|| prod ||inputs||
```

for the finitely many Jordan monomial contexts `C` that occur in the Jordan identity, then

```text
|| ((a*a)*b)*a - (a*a)*(b*a) || <= C eps ||a||^3 ||b||.
```

The missing contextual estimate is the central Layer 2 problem.

## Still-Proven Easy JB-Type Axioms

Even without the Jordan identity, the following estimates are valid for `a,b in A`:

```text
||a*b|| <= (1+C eta)||a||||b||,
a*a = P(a^2) >= -C eta ||a||^2 1,
||a*a|| >= (1-C eta)||a||^2.
```

The only missing epsilon-JB axiom is the approximate Jordan identity.

## Current Status

This note proves the square-root version of the first Effros-Stormer insertion lemma. It does **not** complete the Layer 2 bridge.

The remaining open problem is either:

1. prove a contextual insertion estimate, preferably `O(eta)` but even `O(sqrt(eta))` would give a theorem; or
2. find a counterexample showing that UP almost-idempotence does not force small Jordan identity defect.
